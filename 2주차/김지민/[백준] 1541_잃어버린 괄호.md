### 문제 풀이 핵심 아이디어 :
    1) '-' 기준으로 나눔.
    2) '+' 기준으로 연산을 진행함.
    3) 나머지 '-' 연산을 진행함.

### 코드의 시간복잡도와 그 이유 :
    O(n*m) : for문 '+', '-'을 기준으로 split하여 연산을 진행함.
            '-'를 기준으로 split된 개수 n과 '+' 기준으로 split된 개수 m만큼 for문을 중첩하여 연산함.

### 코드 :
```python
a = input().split('-')

num = []

for i in a :
    sum = 0
    tmp = i.split('+')
    for j in tmp :
        sum += int(j)
    num.append(sum)

n = num[0]

for i in range(1, len(num)) :
    n -= num[i]

print(n)
```
